# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-28

### Added
- Initial release of Cuttlefish
- Type-safe client for Octopus Energy API
- Support for electricity and gas consumption data
- Account information endpoints
- Products and tariffs endpoints
- Grid supply points lookup
- Electricity meter point information
- Configurable retry policy with exponential backoff
- Authentication management
- Typed domain models using opaque types
